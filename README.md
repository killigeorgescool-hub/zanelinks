## 🚀 Deployment

### Option 1: Local Files
- Simply open `index.html` in your browser

### Option 2: Deploy Online (Free)

#### **Netlify** (Recommended)
1. Go to https://app.netlify.com
2. Connect with GitHub/Google
3. Drag & drop your project folder
4. It's deployed! You'll get a custom URL

#### **GitHub Pages**
1. Push the project to GitHub
2. Go to Settings → Pages
3. Select the main branch and root folder
4. Your site is online!

#### **Vercel**
1. Go to https://vercel.com
2. Import your GitHub project
3. Click "Deploy"
4. Done!

#### **Render**
1. Go to https://render.com
2. Create a new "Static Site"
3. Connect your GitHub
4. Deploy in a few clicks

## 🎨 Customization

### Change Colors
In `style.css`, modify the color values:
- Primary blue: `#0066ff`, `#00d4ff`
- Background: gradient `#0a0e27`, `#1a1f3a`

### Change Links
In `index.html`, modify the URLs in the `<a>` tags and add/remove cards as needed.

### Change Icons
Icons come from Font Awesome. Visit https://fontawesome.com for more options.

### Add More Links
Duplicate a `.link-card` section in the HTML and update:
- The URL `href`
- The icon with the Font Awesome class
- The title and description
